# OOP Introduction Explained with a TypeScript Console Application

In this project you are going to follow this <a href="https://www.youtube.com/watch?v=QboWn0NOUA8">video</a> which explains object oriented programming using C# and write the same code in TypeScript.

Let's create a simple program that demonstrates the four main principles of Object-Oriented Programming (OOP): Encapsulation, Inheritance, Polymorphism, and Abstraction. We'll use a `Vehicle` class and a `Car` class that extends `Vehicle`.

## In this example

- `Vehicle` is a class that encapsulates the properties `speed` and `numberOfWheels`, and the methods `accelerate`, `decelerate`, and `getSpeed`.
- `Car` is a subclass of `Vehicle` that inherits its properties and methods, and adds an additional property `brand`. It also overrides the `accelerate` method to provide a different implementation.
- The `startEngine` method in the `Car` class is an example of abstraction. It hides the details of how the engine is started, and provides a simple interface to the client.
- The `myCar` object is an instance of the `Car` class. We can call methods on this object to manipulate its state.
- This is a basic introduction to OOP in TypeScript. There are many more concepts to learn, such as interfaces, abstract classes, and generics. Please let me know if you need help with anything else.
