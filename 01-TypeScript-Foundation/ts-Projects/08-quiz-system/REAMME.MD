# Quiz System

The app will show the students multiple choice questions and promt the user to reply. In the end it will show the students the result of the quiz.


This is a simple TypeScript program that implements a console-based Quiz System. This program uses the `inquirer` library for user interaction. I've added comments to explain what each part of the code is doing:

This program provides a simple text-based quiz where the user is asked multiple choice questions. The `Question` class represents a question with a question text, options, and the correct answer. The `Quiz` class represents the quiz itself, with the questions and the user's score, and a method to start the quiz. The quiz asks each question in turn, and if the user's answer is correct, their score is incremented. At the end of the quiz, the user's score is printed. Note that you'll need to install the `inquirer` package using npm (Node Package Manager) before running this program. You can do this by running `npm install inquirer` in your terminal. Also, this code should be compiled with TypeScript compiler and run with Node.js. Please let me know if you need help with anything else.
