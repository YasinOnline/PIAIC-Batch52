# Word Counter

The user will enter a english paragraph and all that is needed is to just to implement counting characters and words without whitespaces.

This program provides a simple command line interface for counting the number of words and characters in a paragraph. It prompts the user to enter a paragraph, then it counts the number of words by splitting the paragraph at each space and counting the resulting array elements. It counts the number of characters by removing all whitespaces from the paragraph and counting the remaining characters. Note that you'll need to install the `inquirer` package using npm (Node Package Manager) before running this program. You can do this by running `npm install inquirer` in your terminal. Also, this code should be compiled with TypeScript compiler and run with Node.js. Please let me know if you need help with anything else.
