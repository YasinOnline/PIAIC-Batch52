# Currency Converter

The TypeScript console app is used to convert currencies: the users enter a certain amount of money in one currency and set the currency they want to check the monetary value of.

While developing the app, the beginners can master variables, algorithms, loops, if statements, and other TypeScript concepts.

This program provides a simple command line interface for converting currencies. It prompts the user to enter an amount and two currency codes, then it makes a request to the exchange rate API to get the conversion rate between the two currencies, and finally it calculates and prints the converted amount. Note that you'll need to install the `inquirer` and `axios` packages using npm (Node Package Manager) before running this program. You can do this by running `npm install inquirer axios` in your terminal. Also, this code should be compiled with TypeScript compiler and run with Node.js.
