# Simple Command Line Calculator

Develop a simple command line Todo app using TypeScript, Node.js and Inquirer.

This is a simple TypeScript program that uses Node.js and Inquirer to create a command line Todo app:

This program provides a simple command line interface for managing a todo list. It allows the user to add tasks, view all tasks, and mark tasks as completed. The tasks are stored in an array. The `main` function is the entry point of the program, which prompts the user to choose an action. The `addTask`, `viewTasks`, and `completeTask` functions handle the corresponding actions. Note that you'll need to install the `inquirer` package using npm (Node Package Manager) before running this program. You can do this by running `npm install inquirer` in your terminal. Also, this code should be compiled with TypeScript compiler and run with Node.js. Please let me know if you need help with anything else.
