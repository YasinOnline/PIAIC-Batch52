# Learning Object Oriented Programming with TypeScript MyBank Console App

This code demonstrates the following functionalities:

Sample data: It creates two customers and two bank accounts with initial balances.
Show account info: A showAccountInfo function displays the customer name, account number, and balance.
Transactions: The myBankApp function simulates debit and credit operations on the accounts.
Output: The code logs messages to the console, demonstrating the transactions and updated balances.
Remember to install and configure a test runner like Jest (as mentioned in the previous response) to run the unit tests in bankApp.test.ts.
