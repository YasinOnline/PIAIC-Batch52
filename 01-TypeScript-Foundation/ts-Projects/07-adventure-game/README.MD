# Text-Based Adventure-Game in TypeScript and Node.js

This project is not GUI based. It is a console-based game. The video here shows how to develop the game in Java. You will take the requirements of the game from the video and develop the game in TypeScript and Node.js

To create a text-based adventure game in TypeScript and Node.js based on a general understanding of such games. Here's a simple example of how you might structure such a game:

This program provides a simple text-based adventure game where the player can move between different rooms. The `Room` class represents a room with a description and exits to other rooms. The `Game` class represents the game itself, with the current room the player is in and a method to start the game. The game starts in `room1`, and the player can move to `room2` by going north. If the player tries to go in a direction that doesn't lead to another room, they are told that they can't go that way. Note that you'll need to install the `inquirer` package using npm (Node Package Manager) before running this program. You can do this by running `npm install inquirer` in your terminal. Also, this code should be compiled with TypeScript compiler and run with Node.js. Please let me know if you need help with anything else.
